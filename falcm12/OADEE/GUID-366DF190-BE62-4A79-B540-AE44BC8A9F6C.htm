New and Changed Features for Release 12 (11.12.x.0.0)

The Oracle Fusion Applications Developer's Guide Release 12 (11.12.x.0.0) includes the following new and changed features for this document.

  • The nomenclature “11.12.x.0.0”, where “x” is a number, is used to indicate the release and patch releases for which the guide is applicable.

  • The Implementing Attachments chapter has been extensively reorganized and updated.

  • A new subsection, About Configuring the Attachment Link Component UI, has been added to the Configuring Attachments Components Propertiessection.
