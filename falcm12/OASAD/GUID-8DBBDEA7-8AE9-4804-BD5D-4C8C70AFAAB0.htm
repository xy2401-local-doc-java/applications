1 Overview and Roles

This section provides an overview of the administrative roles likely to use Fusion Applications Lifecycle Management.
The following topic is discussed:

  • Roles and Key Tasks

1.1 Roles and Key Tasks

The following list contains the administrative users who might find this guide useful:

  • System administrator

  • Network engineer

  • Database administrator

  • Oracle Identity Management and security specialist

  • Oracle Fusion Applications technical lead or system administrator

  • Support technician

The mapping of deployment tasks and responsibilities to organizational roles may vary across different types of enterprises depending on corporate culture and organizational structure.

1.1.1 System Administrator

The system administrator could be responsible for:
  • The critical hardware and operating system layers of the Oracle Fusion Applications deployment
  • Ensuring that all hardware meets with organizational standards

  • Ensuring that operating systems are configured in accordance with Oracle Fusion Applications requirements

  • Overall systems testing

  • Coordinating all necessary system and subcomponent migrations from testing to QA

  • Production platforms

The primary tasks may be:

  • Procuring and setting up hardware

  • Installing and configuring operating systems

  • Filling out the following tabs in theOracle Fusion Applications Installation Workbook-

    • Topology tab: All

    • Storage tab: All

    • Network - Ports tab: All

1.1.2 Network Engineer

The network engineer may be responsible for:
  • The setup and ongoing maintenance and monitoring of all components that facilitate communication among computers within the enterprise computing infrastructure
  • The configuration and setup of machine interfaces such as hosts tables, network cards, network interfaces, IP address allocations, and network equipment such as switches, routers, gateways, and load balancers

The primary tasks could include the following:

  • Setting up and configuring machine interfaces and network equipment

  • Monitoring and maintaining the network infrastructure

  • Filling out the following tabs in the Oracle Fusion Applications Installation Workbook-

    • Environment tab: Email Server table

    • Environment tab: Web Proxy table

    • Topology tab: Topology table

    • Network - Virtual Hosts tab: All

    • Storage tab: All

    • Network - Ports tab: All

1.1.3 Database Engineer

The database engineer tasks could include:
  • Managing the database layer of the Oracle Fusion Applications deployment
  • Filling out the Database tab in the Oracle Fusion Applications Installation Workbook

1.1.4 Oracle Identity Management and Security Specialist

The security specialist responsibilities could be:
  • Create and maintain policies to protect corporate data and computing resources from a variety of real and potential threats
  • In the context of Oracle Identity Management, the tasks may include creating and managing the enterprise computing components that authenticate, authorize, and account for individual access to computing resources and systems.
  • Control the corporate LDAP directories and their associated AAA systems
  • Oracle Fusion Applications deployments require creating and configuring identity and policy stores that may differ from pre-existing Oracle Identity Management components in the enterprise. As such, the security specialist should coordinate the effort to install and manage Oracle Identity Management components bearing in mind the effects the new components may have on existing components in the enterprise.

The primary tasks for this role may include:

  • Creating and managing authentication and authorization enterprise computing components

  • Creating and configuring identity and policy stores for Oracle Identity Management, in conjunction with existing Oracle Identity Management components

  • Managing security certificates

  • Managing password policies (functional security)

  • Managing roles and responsibilities (functional setup)

  • Managing system account password maintenance policie

  • Setting up the database (Vault, if applicable)

  • Filling out the following tabs in the Oracle Fusion Applications Installation Workbook-

    • Provisioning tab: Oracle Identity Management Products table

    • Identity Management tab: All

    • SSL and Certificates tab: All

    • Topology tab: DMZ column in the Topology table

    • Environment tab: Web Proxy table

1.1.5 Oracle Fusion Applications Technical Lead or System Administrator

The Oracle Fusion Applications technical lead responsibilities might include:
  • Working closely with other IT specialists, including the database, network, and system administrators as they install and configure the Oracle Fusion Applications infrastructure

  • Coordinating hands-on management of low-level tasks to be completed during technical setup and validation

The primary tasks could be:

  • Managing technical setup and validation tasks during Oracle Fusion Applications deployment

  • Managing system interfaces

1.1.6 Support Technician

The support technician primary responsibilities might be:
  • Assisting end users with any computing applications in the enterprise
  • Covering the Oracle Fusion Applications deployment system

Many organizations have technical and functional areas of specialization within the support technician role.
