Preface

This guide describes how to administer the Oracle Fusion Applications environment, including how to start and stop components, handle basic security, monitor the environment, perform tuning, handle logging, troubleshooting, and backup and recovery.

Audience

This document is intended for administrators of the Oracle Fusion Applications On-Premise environment.
