Welcome Page for the configApps File

This screen introduces you to the Oracle Business Analytics Applications Suite configuration process.

Before you run the configApps.bat/sh file, you must complete these prerequisites:

  • Run the Business Analytics Applications Suite installer. See Installing Oracle BI Applications Using the Business Analytics Applications Suite Installer

    .

  • Apply Oracle Fusion Middleware platform patches. See Applying the Oracle Fusion Middleware Platform Patch.

    Do not proceed with the configuration unless you have applied the platform patches. If you have not applied the patches, exit the configuration utility and apply the patches. Then, re-run the configApps.bat/sh file.
