Exporting Topology Settings from Original ODI Repository

You need to export the topology settings from the Original ODI Repository that were created using the Oracle Business Analytics Suite RCU.

To export the topology settings from original ODI repository:
  1. Launch the ODI Studio client and connect to the original ODI repository.
  2. Navigate to the Topology tab. From the Connect Navigator (Topology icon dropdown on the top right side of the navigator pane), select Export. As part of the procedures described below, you will export files to a local directory.

    Description of the illustration GUID-C66268AC-ADD4-4624-B5F5-6BDF35152F32-default.gif
  3. Launch the Smart Export wizard from the Export Selection dialog.

    Description of the illustration GUID-664665E6-35F7-4474-9146-D0ABC7A9E46D-default.gif
  4. Drag the below logical schemas Objects to be Exported window as shown below. Provide a meaningful name for the export file. Click Export.

    Description of the illustration GUID-E73BD9FA-19CC-453A-93B6-39B8A6B876F2-default.gif
    This will export the logical and physical topology including assigned Data source Num ID values and database connect details.
