102/119

Configure Ports

Use this screen to configure the ports for the Oracle Business Intelligence system.

Field Description

Auto Port Configuration

Select this option if you want Oracle Business Analytics Applications Suite installer to configure the ports for you.

Specify Ports Using Configuration File

Select this option to use an existing configuration file to configure the ports. Optionally, click Browse to specify its location. Click View/Edit File to view or edit the file.

The sample configuration file is named staticport.ini and is provided in the Disk1\stage\Response directory of the Business Analytics Applications Suite installer.
