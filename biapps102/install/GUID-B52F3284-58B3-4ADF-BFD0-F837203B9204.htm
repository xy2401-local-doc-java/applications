100/119

Business Analytics Warehouse Schema

Use this screen to specify connection details for the Oracle Business Analytics Warehouse schema.

Field Description

Database Type

Specify the type of database that hosts the Oracle Business Analytics Warehouse schema. The only supported database type is Oracle.

Connect String

Specify the connect string used to access the Oracle Business Analytics Warehouse schema.

Use the format:

host:port:service_name

Business Analytics Warehouse Schema Username

Specify the schema name for the Oracle Business Analytics Warehouse schema.

Business Analytics Warehouse Schema Password

Specify the password for the Oracle Business Analytics Warehouse schema.
