Payroll PYRL-Based Fusion Security

The Payroll List is determined at user sign-on via one or more initialization blocks.

Name: OBIA_PAYROLL_DATA_SECURITY

Display Name: Payroll PYRL-based Fusion Security

Initialization Blocks

Variable Name Initialization Block Name

HR_SEC_PAYROLL_LIST____FUSN

HR - Payroll - Payroll List(Fusion)

Data Security Role Filters

The Payroll - based Security is applied depending on the roles the user is granted, and when it is applied it is supported by the following HR logical facts and dimensions:
