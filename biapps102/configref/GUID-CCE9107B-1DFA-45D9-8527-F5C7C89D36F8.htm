Compensation Manager (Secured by Legislative Data Group List) AU BI Data

The Compensation Manager LDG list is determined at user sign-on via one or more initialization blocks.

Name: OBIA_AU_HCM_CMPMGR_LDG_DATA_SECURITY

Variable Name Initialization Block Name

HR_SEC_LDG_CMPMGR_LIST
