Payroll LDG-Based Fusion Security

The Compensation Analyst LDG list is determined at user sign-on via one or more initialization blocks.

Display Name: Payroll LDG-based Fusion Security

Name: OBIA_LEGISLATIVE_DATA_GROUP_DATA_SECURITY

Initialization Blocks

Variable Name Initialization Block Name

HR_SEC_LDG_LIST____FUSN

HR - Payroll - Legislative Data Group List (Fusion).

Data Security Role Filters

The Legislative Data Group-based Security is applied depending on the roles the user is granted, and when it is applied it is supported by the following HR logical facts and dimensions:
