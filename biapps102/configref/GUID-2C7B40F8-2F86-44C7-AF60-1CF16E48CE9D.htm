1 Functional Configuration Task Reference

This is a reference section that contains Help topics for Informational Tasks in Functional Setup Manager (FSM). Informational Tasks display conceptual information, or display configuration steps that are performed in tools external to FSM (for example, in Oracle Data Integrator (ODI), or Oracle Business Intelligence Administration Tool).

The Help topics are displayed in FSM when you click Go to Task for an Informational Task, or you click a Help icon for additional information about an FSM.

To perform the functional configuration tasks, see Performing Functional Configuration in Oracle Business Intelligence Applications Configuration Guide.

To set up security, see Setting Up Security with Functional Setup Manager in Oracle Business Intelligence Applications Security Guide.
