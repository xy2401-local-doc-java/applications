Creating a Backup of the 11.1.1.10.1 ODI Repository

Follow these instructions to create a backup of the 11.1.1.10.1 ODI repository.

Use the Oracle Database Export and Import utility to export the ODI Repository for Oracle BI Applications 11.1.1.10.1 from the existing schema and import it into another schema. This will allow you to connect to the 11.1.1.10.1 ODI repository in case you need to reference it.

Create a connection in ODI to the 11.1.1.10.1 Backup ODI Repository which is now in a new database schema.

To create a new connection in ODI Studio to the back-up of the ODI Repository for 11.1.1.10.1:
  1. Configure the connection to use the same ODI user you used before.
  2. Configure the connection as a Master Repository connection.
  3. Connect to the repository. Click Topology, click Repositories, and select BIAPPS_WORKREP.
  4. Edit the repository to change the database user details used by the Work Repository to use the new schema.
