Users REST Endpoints

Users
The user resource is used to create, update, delete, or view users in Oracle Applications Cloud.
Create a user
Method: post
Path: /hcmCoreSetupApi/scim/Users
Delete a user
Method: delete
Path: /hcmCoreSetupApi/scim/Users/{id}
Get a user
Method: get
Path: /hcmCoreSetupApi/scim/Users/{id}
Get all users
Method: get
Path: /hcmCoreSetupApi/scim/Users
Replace a user
Method: put
Path: /hcmCoreSetupApi/scim/Users/{id}
Update a user
Method: patch
Path: /hcmCoreSetupApi/scim/Users/{id}
