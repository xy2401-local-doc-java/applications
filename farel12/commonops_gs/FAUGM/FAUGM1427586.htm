Managing Scheduled Processes for Assigning Performance Goals: Procedure

The process of assigning performance goals using goal plan sets, goal plans, and mass assignment requests is an Enterprise Scheduler Service process. HR specialists submit and run the process on the Manage Goal Scheduled Process page in the Goals work area.

When you submit a goal plan set from the Manage Goal Plan Sets page, a goal plan from the Manage Goal Plans page, or a mass assignment request from the Mass Assign Goals page, the Manage Goal Scheduled Process page appears automatically.

Managing Scheduled Processes

Follow these steps:

  1. Select Navigator - About Me - Goals.

  2. In the Related Links panel on the Tasks tab, click Manage Goal Scheduled Process. The Manage Goal Scheduled Process page appears.

  3. Search for and select a process name.

  4. Run the process as soon as possible or at a scheduled time.

    Note: By default, the process runs as soon as possible if you submit the process without selecting a schedule.

  5. Click Advanced and select Using a schedule to specify a date and time when you want the process to run. Or set up a recurring schedule for the process to run at the selected frequency.

After the process completes, the application assigns performance goals, goal plans, and goal plan sets as included in the request to all workers selected in the request.

Manage Goal Scheduled Process Parameters for Goal Management

HR specialists can schedule and run the process to assign performance goals through goal plans, goal plan sets, or mass assignment requests. They can manage scheduled processes for performance goals on the Manage Goal Scheduled Process page. Use the Goals work area to open the Manage Goal Scheduled Process task in the Related Links panel on the Tasks tab.

Process Parameters

Process Type

Identifies the request being processed. If you rerun an existing process, it doesn't effect any goals or goal plans assigned earlier to any workers.

You can select the following process types:

  • Mass assign goals to submit a process related to a mass assignment request for performance goals.

  • Assign goal plans to submit a process related to goal plans. If you select this process type, you can also specify the Effective Date attribute. This attribute returns workers included in the goal plan and having a hire or start date same as or earlier than the specified effective date. If you don't specify a value for the effective date, the application uses the current date as the effective date. For scheduled processes, the current date is used as the effective date every time the process runs.

    The page also displays the start and end dates of the goal plan included in the process.

  • Assign goal plan sets to submit a process related to goal plan sets. If you select this process type, you can also specify the Effective Date attribute. This attribute returns workers included in the goal plan set and having a hire or start date same as or earlier than the specified effective date. If you don't specify a value for the effective date, the application uses the current date as the effective date. For scheduled processes, the current date is used as the effective date every time the process runs.

    The page also displays the start and end dates of the goal plan set included in the process.

Process Name

Displays the process name that identifies the specific goal plan set, goal plan, or mass assign request being processed.
