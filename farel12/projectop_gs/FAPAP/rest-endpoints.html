Project Progress
An object that includes the attributes that are used to store the progress data for a project.
Get the progress of a project
Method: get
Path: /projectProgress/{ProjectId}
Get the progress of all projects
Method: get
Path: /projectProgress
Update the progress of a project
Method: patch
Path: /projectProgress/{ProjectId}
Project Progress/Task Progress
An object that includes the attributes that are used to store the progress data for a project task.
Get the progress of a project task
Method: get
Path: /projectProgress/{ProjectId}/child/TaskProgress/{TaskId}
Get the progress of all tasks of a project
Method: get
Path: /projectProgress/{ProjectId}/child/TaskProgress
Update the progress of a project task
Method: patch
Path: /projectProgress/{ProjectId}/child/TaskProgress/{TaskId}
Projects
The project resource is used to view, create, or modify a project. A project is the effort and resources required to achieve a significant business objective within a specific, usually finite, time frame.
Create a project
Method: post
Path: /projects
Get a project
Method: get
Path: /projects/{ProjectId}
Get all projects
Method: get
Path: /projects
Update a project
Method: patch
Path: /projects/{ProjectId}
Projects/Project Customers
An object that includes the attributes which are used to store values while creating or updating the customers for a project. This represents the name of the customer with whom an agreement has been made on the contract or project.
Create a project customer
Method: post
Path: /projects/{ProjectId}/child/ProjectCustomers
Get a project customer
Method: get
Path: /projects/{ProjectId}/child/ProjectCustomers/{ProjectPartyId}
Get all customers for a project
Method: get
Path: /projects/{ProjectId}/child/ProjectCustomers

Projects/Tasks
The task resource includes the attributes that are used to store values while creating or updating project tasks. Tasks are units of project work assigned or performed as part of the duties of a resource. Tasks can be a portion of project work to be performed within a defined period by a specific resource or multiple resources.
Create a project task
Method: post
Path: /projects/{ProjectId}/child/Tasks
Delete a project task
Method: delete
Path: /projects/{ProjectId}/child/Tasks/{TasksUniqID}
Get a project task
Method: get
Path: /projects/{ProjectId}/child/Tasks/{TasksUniqID}
Get all tasks for a project
Method: get
Path: /projects/{ProjectId}/child/Tasks
Update a project task
Method: patch
Path: /projects/{ProjectId}/child/Tasks/{TasksUniqID}
