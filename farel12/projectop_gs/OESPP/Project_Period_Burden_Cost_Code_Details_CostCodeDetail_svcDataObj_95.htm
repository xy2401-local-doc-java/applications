Attributes

The following table describes attributes in the service data object.

Name Type Description

BurdenCostAmount

{http://xmlns.oracle.com/adf/svc/types/}AmountType

Cost for the burden cost code. Burden costs are legitimate costs of doing business that support raw costs and cannot be directly attributed to work performed.

BurdenCostRate

decimal

The calculated rate used to determine the burden cost for a planning resource. The rate is the burden cost divided by the quantity. The rate is blank for amount-based resources.

BurdenExpenditureType

string

Expenditure type associated with the burden cost code.

BurdenMultiplier

decimal

Multiplier calculated as a result of the burden precedence. The multiplier is the burden cost divided by the raw cost.

CostCode

string

Name of the burden cost code, which is a classification of overhead costs. A burden cost code represents the type of burden cost you want to apply to raw cost.

Precedence

long

Sequential order that each burden cost code is applied to the running total.
