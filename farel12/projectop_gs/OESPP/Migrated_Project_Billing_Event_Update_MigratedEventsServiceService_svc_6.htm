Migrated Project Billing Event Update

Service to operate on migrated project billing events to update the event amount in contract, invoice and revenue, project, and ledger currencies. Use migrated events for all types of contract lines. For example, an invoice-applicable migrated event for the ITD invoiced amount for a fixed-price contract line invoiced on a milestone basis, and a revenue-applicable migrated event for the ITD revenue recognized amount for a time and material contract line recognized as incurred.

Life Cycle Status: Active
