Task Detail Update Criteria Version 2

The task data used by the project task service for updating tasks in a project. Data examples include task name, task number, and validation indicators that specify what task validations will take place during task updates.

QName: {http://xmlns.oracle.com/apps/projects/foundation/projectDefinition/publicService/maintainProjectV2/}ProjectTask
