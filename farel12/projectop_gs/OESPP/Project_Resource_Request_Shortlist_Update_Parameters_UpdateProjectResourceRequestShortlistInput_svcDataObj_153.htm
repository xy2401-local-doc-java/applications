Project Resource Request Shortlist Update Parameters

The input parameters for the project resource request service to add or remove resources from a project resource request shortlist.

QName: {http://xmlns.oracle.com/apps/projects/resourceManagement/request/resourceRequestPublicService/}UpdateProjectResourceRequestShortlistInput

Relationships

The following table describes how this service data object is related to other service data objects or business object services.

Relationship Type Object or Service

Handled by

Project Resource Request

Attributes

The following table describes attributes in the service data object.

Name Type Description

RemoveResourceFromShortlistFlag

boolean

Indicator to determine whether to add or remove the resource from a project resource request shortlist.

ResourceRequestId

long

Identifier of project resource request.

ShortlistId

long

Identifier of the shortlist resource that represents a unique combination of a project enterprise labor resource and a project resource request.

ShortlistResourceEmail

string

E-mail of the project enterprise labor resource who is added to or removed from a project resource request shortlist.

ShortlistResourceHCMPersonId

long

Identifier of the HCM person associated with the project enterprise labor resource who is added to or removed from a project resource request shortlist.

ShortlistResourceId

long

Identifier of the project enterprise labor resource who is added to or removed from the project resource request shortlist.

ShortlistResourceName

string

Name of the project enterprise labor resource who is added to or removed from a project resource request shortlist.
