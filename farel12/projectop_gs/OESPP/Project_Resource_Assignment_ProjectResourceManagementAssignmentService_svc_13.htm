Project Resource Assignment

Service to operate on an assignment of a resource to a project. Project resource assignment involves information such as resource name, project name, project role, and assignment dates. For example, Amy Marlin is assigned to the Implementation project as a Technical Lead from June 2 to June 30, 2014.

Life Cycle Status: Active
