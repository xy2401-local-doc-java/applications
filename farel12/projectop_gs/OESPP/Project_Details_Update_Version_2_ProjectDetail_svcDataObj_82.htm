Project Details Update Version 2

A project represents the effort and resources required to achieve a significant business objective within a specific, usually finite, time frame. Projects are used to plan, track, manage, and control both the financial aspects and the work effort required.

Keys

  • Primary Key

ProjectName

string

Name of the project. If you don't provide other project identifiers, then ProjectName identifies the project to be updated. If you provide other project identifiers, then you can update the project name to ProjectName value.

ProjectNumber

string

Number of the project to be updated. You cannot update project number if the project has existing billing events or cost transactions that reference the project number or if the project has a budget enabled for budgetary control.
