InvokeMenu

Description

Invoke Smart View Oracle BI EE extension menu.

Syntax

Sub InvokeMenu(

menuID As String

)

Parameters

menuID: The ID of the menu items. Valid values are listed in Table 16-2.

Table 16-2 Oracle BI EE Menu Items and IDs

Menu ID
View Designer ViewDesigner
Publish View PublishView
Refresh Refresh
Edit Prompts EditPrompts
Edit Page Prompts EditPage
Copy CopyView
Paste PasteView
Delete DeleteView
Mask Data MaskView
Mask Document Data MaskDocumentView

Example

Sub TestMenuInvoke()

    Dim obiee As IBIReport
    Set obiee = New SmartViewOBIEEAutomation
    
    obiee.InvokeMenu "ViewDesigner"

End Sub
