Editing Essbase Cell Comments

Be sure to complete the steps in Enabling Essbase Cell Comments.

Note:

The procedure in this topic shows you how to edit Oracle Essbase cell comments using the View Comments command and the Comment Edit dialog box in Oracle Smart View for Office. You can also edit cell comments directly in the grid, without using the Smart View interface elements in this topic.

To edit Essbase cell comments:

  1. If you have not already done so, open a workbook containing a grid, connect to an Essbase provider, and click Refresh.
  2. Click View Comments in the Essbase ribbon.

    Figure 5-15 View Comments Command in Essbase Ribbon



    Tip:

    The View Comments command is enabled only when there are comments in the sheet.

    The Comment Edit dialog box is launched. The example in Figure 5-16 shows two comments in the sheet which can be edited.

    Figure 5-16 Comment Edit Dialog Box



  3. To edit the text in a row, first click in the row to highlight the editable text; for example, click in the "Oregon Coast" row to highlight it, as shown in Figure 5-17

    Figure 5-17 Row Highlighted, Ready for Editing in Comment Edit Dialog Box



  4. Edit the text as required; in this example, change "Oregon Coast" to "Oregon".

    Figure 5-18 Changed Text in Comment Edit Dialog Box



  5. Click Apply Changes and note the change in the grid.

    In Figure 5-19, cell C5 still shows as a commented cell, and will until the sheet is refreshed.

    Figure 5-19 Grid After Clicking Apply Changes; Cell C5 Still Commented



  6. Click Refresh; the comment style is cleared.

    Figure 5-20 Grid After Refresh, Cell Style in Cell C5 Is Cleared; Member is In Sync with Server
