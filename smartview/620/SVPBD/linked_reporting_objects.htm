Linked Reporting Objects

Data source types: Oracle Essbase, Oracle Analytics Cloud - Essbase

A linked reporting object is a cell note, external file, or URL that is linked to a data cell in an Essbase database, and which can be retrieved by Oracle Smart View for Office users in Excel.

You can set a cell style to identify cells that are associated with linked reporting objects. See Cell Styles.
