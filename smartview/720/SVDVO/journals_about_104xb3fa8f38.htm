About Oracle Journals for Financial Management Functions

The VBA functions in this chapter support operations when connected to an Oracle Hyperion Financial Management data source.

To use the VBA functions in this chapter, Financial Management 11.1.2.4.204 with the 11.1.2.4.204 PSE 25575478 is required. Additionally, you must install Oracle Smart View for Office 11.1.2.5.700 and the Oracle Journals for Financial Management extension that is included with the 11.1.2.5.700 release.

You must complete the following procedures in order before you begin working with the VBA functions for Oracle Journals for Financial Management.

  1. Registering the Oracle Journals VBA Functions Using RegAsm

  2. Preparing to Work with Oracle Journals for Financial Management Functions

  3. Instantiating an Oracle Journals for Financial Management Extension Object
