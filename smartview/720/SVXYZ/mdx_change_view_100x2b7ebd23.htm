Changing the Query List Display

Use the Change View button to change how queries are listed in the in the left pane of the Manage and Execute Queries dialog box.

The query list display can be toggled from a flat alphabetical list of all queries associated with the connection, to a tree view where queries are listed by their associated applications and database.

To change the query list display:
  1. In Excel, connect to an Oracle Essbase or Oracle Analytics Cloud - Essbase data source.
  2. From the Essbase ribbon, select Manage and Execute Queries
  3. In the Manage and Execute Queries dialog box, click the Change View button, .

    The default display is a flat alphabetical list. Clicking changes the query list to a tree view, where queries are listed by their associated applications and databases, as shown in Changing the Query List Display

  4. Optional: Click again to toggle the view back to an alphabetical list.
