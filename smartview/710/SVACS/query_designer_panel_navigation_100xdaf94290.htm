Table 9-1 Query Designer Panel Navigation

Keys Action
F6 Switch between Query Designer and Office components
Tab Moves through the dimensions on the Query Designer
Ctl+Enter, when selected on a row or column dimension Launches the Member Selection dialog box and highlights the corresponding dimension on the query sheet
Ctl+Enter, when selected on a POV dimension Adds the selected dimension to the Row dimensions group in the Query Designer
