Importing Reporting and Analysis Documents

Using Oracle Smart View for Office, you can import Oracle Hyperion Reporting and Analysis documents into Microsoft Excel, Word, or PowerPoint.

Note:

In this topic, references to Oracle Hyperion Financial Reporting apply to both the on-premises and Oracle Planning and Budgeting Cloud versions of Financial Reporting.
