Data source types: Oracle Essbase, Oracle Analytics Cloud - Essbase, Oracle Hyperion Planning, Oracle Planning and Budgeting Cloud, Oracle Hyperion Financial Management

Simple Grids in Essbase

In Essbase (starting with the 11.1.2.1.102 release) or Oracle Analytics Cloud - Essbase and Oracle Smart View for Office, free-form behavior is such that the POV members are placed on the first row of an Excel sheet. Column and row dimension members are placed on the grid starting after the POV row rows. This means that all the POV members become part of the grid.
