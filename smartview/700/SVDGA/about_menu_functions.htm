About Menu Functions

VBA menu functions are identical to the equivalent commands on the Oracle Smart View for Office menu and ribbon. The requirements for the menu functions are the same as those for the menu commands. For example, if you must be logged in to an Oracle Essbase server to use a menu command, then you must also be logged in to an Essbase server to use the equivalent VBA command.
