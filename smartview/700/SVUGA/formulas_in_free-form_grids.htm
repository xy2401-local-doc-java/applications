Formulas in Free-Form Grids

Data source types: Oracle Essbase, Oracle Analytics Cloud - Essbase, Oracle Hyperion Planning, Oracle Planning and Budgeting Cloud, Oracle Hyperion Financial Management

You can enter Excel formulas in cells that can contain comments.

Note:

Excel formulas are not saved to the provider server in the free-form worksheet. For example, if you use the Save Ad Hoc Grid command with a Planning free-form or ad hoc grid, the formulas are not saved. However, you can retain formulas in a worksheet by performing a Save in Excel and saving the workbook.
