About Free-Form Mode

In ad hoc analysis, if you are familiar with the dimensions and members of your database, you can use free-form mode by typing dimension and member names directly into cells. You can still use the POV, member selection, and other ad hoc operations in free-form grids.

The components of Oracle Smart View for Office grids are described in Table 23-1.

Table 23-1 Smart View Grid Components

Grid Component Description

Row Dimension

A dimension or member placed down one column across one or more rows in a worksheet

Column Dimension

A dimension or member placed on a row across one or more columns in a worksheet

Page Dimension

A dimension that applies to the entire page (Oracle Essbase only)

Comments

Text added by the user

Data Region

Areas of the grid that contain data for dimensions or members

Blank Region

Areas of the worksheet that contain no entries
