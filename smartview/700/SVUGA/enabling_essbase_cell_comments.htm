Enabling Essbase Cell Comments

To enable cell comments in Oracle Essbase:

  1. In the Options dialog box, perform these tasks:
    1. In Member Options, select Enable Enhanced Comment Handling.

      Note:

      The Preserve Formulas and Comments in ad hoc operations (except pivot) option must be selected to enable the Enable Enhanced Comment Handling option.

    2. In Formatting, select Use Cell Styles.
    3. In Cell Styles, expand Common, and then select the check box next to Comment Cells. Next, right-click Comment Cells and define a Font, Background, or Border for cells containing cell comments.
  2. Click OK.

    Optionally, click the arrow in the OK button and choose either Save as Default Options or Apply to All Sheets. These options are described in "Sheet Options" in Smart View Options.

    You are now ready for the steps in Viewing Essbase Cell Comments in the Sheet
