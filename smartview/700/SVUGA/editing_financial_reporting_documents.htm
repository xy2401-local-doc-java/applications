Editing Financial Reporting Documents

Note:

In this topic, references to Oracle Hyperion Financial Reporting apply to both the on-premises and Oracle Planning and Budgeting Cloud versions of Financial Reporting.

To edit Financial Reporting documents in Excel, Word, and PowerPoint:

  1. Open the Financial Reporting document to edit.
  2. From the Oracle Smart View for Office ribbon, click the arrow in , then select Reporting and Analysis Document, and then Edit.

    The Import Workspace Document wizard is displayed.

    Note:

    Some wizard screens do not apply to some documents.

  3. If the Preview User Point of View screen is displayed, preview the current POV or change the members of the POV.

    Note:

    To display this screen, in Oracle Hyperion Enterprise Performance Management Workspace preferences, select Preview for User Point of View.

  4. Optional: In documents that contain prompts, to change the default value, in Respond to Prompts, make a selection for prompts, and click Next.

    Note:

    Respond to Prompts is displayed only if the document contains prompts.

  5. To change the POV, in Preview from Grid POV select a POV.
  6. Click Finish.
