Working with Crystal Ball EPM Workbooks

Permissions set by the Oracle Hyperion Enterprise Performance Management Workspace administrator govern simulation and forecasting workbooks operations that you can perform from the Smart View Panel.

To work with data in a Oracle Crystal Ball Enterprise Performance Management workbook:

  1. From the Oracle Smart View for Office ribbon, select Panel.
  2. In the Smart View Panel, click and select Simulation Workbook.
  3. Click and if requested, log in to the Crystal Ball EPM repository. A tree list containing the workbooks for which you have permission is displayed.
  4. Double-click a workbook to open.
  5. Perform Crystal Ball EPM operations as described in the Crystal Ball EPM product documentation.

    Oracle recommends keeping the Crystal Ball EPM model on a worksheet separate from data source worksheets.

  6. Click Submit Data if needed.
