Composite Forms

Data source types: Oracle Planning and Budgeting Cloud

If you are unfamiliar with Oracle Planning and Budgeting Cloud, see the Oracle Planning and Budgeting Cloud documentation and videos available on the Oracle Help Center.

To open a Oracle Hyperion Planning composite form:

  1. Connect to a Planning data source that contains composite forms.
  2. From the Connections tree list, double-click a composite form (indicated by ).

    The composite form opens in a new Excel workbook with each subform displayed in a separate worksheet.

Oracle Smart View for Office supports Planning master composite forms.
