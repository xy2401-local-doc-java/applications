Administering Extensions When Using Mozilla Firefox

When using Mozilla Firefox, keep in mind the following:

  • Firefox prevents executable files from being copied and installed from the Internet. This results in a failure while attempting to install an .SVEXT extension package with Firefox by selecting the Open with option. As a result, when prompted to save or run the file, select the Save option, and then double-click the downloaded file to install the extension .

  • When using Firefox as your web browser to launch Oracle Smart View for Office, and Firefox opens a browser tab in with an unresolved URL of oraclesv:\\, enable the extension for Smart View from the Extensions tab of Firefox Add-ons Manager and restart Firefox.
