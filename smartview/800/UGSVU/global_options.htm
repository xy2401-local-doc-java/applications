Global Options

Global options are options that apply to the entire current workbook, including any new worksheets added to the current workbook, and to workbooks that are created later.

Changes to global option settings also affect existing worksheets and workbooks, and other Office documents.

The following are global options:

  • Advanced Options

  • Extensions

  • Cell Styles

Note:

Use the OK button when saving global options. Changes to global options become the default settings for all existing and new worksheets and workbooks so it is not necessary to use Save as Default Options or Apply to All Worksheets with global options. Save as Default Options and Apply to All Worksheets are used with the sheet level option settings only.
