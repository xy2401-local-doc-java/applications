Troubleshooting Connecting to the Oracle BI EE Server

You can troubleshoot the connection to the Oracle Business Intelligence Enterprise Edition server: .

When troubleshooting, note the following behaviors:

  • When attempting to connect from Oracle Smart View for Office to Oracle BI EE while the Oracle BI EE server is down, an error message is displayed. However, after clicking OK in the error message, the Connect to Data Source dialog box is erroneously displayed. Click Cancel to close the dialog box.

  • When trying to insert a view after a connection timeout has occurred, the Connect to Data Source dialog box is displayed. If you click Cancel, the dialog box is relaunched. Click Cancel and the error message is displayed.
