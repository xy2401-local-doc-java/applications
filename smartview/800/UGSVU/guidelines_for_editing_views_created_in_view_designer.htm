508/665

Guidelines for Editing Views Created in View Designer

  • Only views created and inserted from the View Designer in Oracle Smart View for Office can be edited using the View Designer.

  • Views created in Smart View may also be edited in Oracle Business Intelligence Enterprise Edition. However, views created in Smart View and then further edited in Oracle BI EE cannot afterward be edited in View Designer.

    Note:

    To edit views created in Oracle BI EE, use the Oracle BI EE application (see Editing an Analysis in Oracle BI EE).

  • When you use View Designer to edit a table view or pivot table view, Smart View performs a replace operation, effectively replacing the view on the sheet with the newly-edited view; that is, the table on the sheet is deleted and a new table is inserted. If the new table has more rows or columns than the previous table, any content or formulas that you may have added to the sheet prior to editing the view in View Designer may be removed and you will need to add them again.
