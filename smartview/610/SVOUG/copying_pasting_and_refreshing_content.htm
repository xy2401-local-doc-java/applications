Copying, Pasting, and Refreshing Smart View Content

The following Oracle Smart View for Office content can be copied and pasted using the Copy and Paste buttons in the Smart View ribbon, and then refreshed using Smart View Refresh.

  • Ad hoc—all applicable providers

  • Functions—all applicable providers

  • Oracle BI EE content

Other content in Smart View will be pasted as static text, including content from Oracle Hyperion Financial Management forms, Oracle Hyperion Planning forms, and Excel pivot tables.
