Working with Forms in Excel

Forms are grid displays in which you can enter data into the database from Excel and view and analyze data or related text. Certain dimension member values are fixed, giving you a specific view into the data.

Using Oracle Smart View for Office, you can work with Oracle Hyperion Planning, Oracle Planning and Budgeting Cloud, Oracle Financial Consolidation and Close Cloud, and Oracle Hyperion Financial Management forms in Excel.

Note:

Excel worksheets are always protected to prevent entering data for read-only cells. Therefore, some Excel functions, such as AutoSum and F9, are disabled.

In forms opened in Smart View:

  • You can modify data values but not the form structure in forms.

  • Values submitted to the database from Excel must be non-formatted data.

  • If a form is currently loaded in Excel and the administrator changes the form definition on the server side, Oracle recommends that you close the form and reload it. This action ensures that the newest form definitions are displayed.

Customizations made to forms are preserved when you save or refresh only if they are made outside the grid or if they are made to thousands and decimal separators.
